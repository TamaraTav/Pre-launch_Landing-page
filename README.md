# Typemaster Pre-launch Landing Page

A modern, responsive landing page for the Typemaster keyboard pre-launch campaign. Built with HTML and CSS, featuring a clean design and full accessibility support.

## Features

- **Responsive Design**: Fully responsive layout that works seamlessly on mobile, tablet, and desktop devices
- **Accessibility**: Includes ARIA labels, semantic HTML, and proper alt text for all images and SVG icons
- **Modern UI**: Clean and professional design with smooth transitions and hover effects
- **Performance Optimized**: Lightweight code with optimized assets
